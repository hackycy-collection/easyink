# 竞品分析报告: report-designer 数据表格

> 分析对象: [xinglie/report-designer](https://xinglie.github.io/report-designer/)
> 分析日期: 2026-04-08

---

## 1. 整体架构

### 1.1 设计器布局

采用经典的五区布局:

```
+---------------------------------------------------+
|                  顶部工具栏 (物料 + 操作)             |
+---------------------------------------------------+
|                  二级工具栏 (快捷操作)                |
+------+-----------------------------+------+-------+
| 数据源 |                             | 结构树 | 属性  |
| 面板  |        画布区域               | 面板  | 面板  |
|      |                             |      |       |
+------+-----------------------------+------+-------+
|                 概览图 / 动画 / 历史记录              |
+---------------------------------------------------+
|                    状态栏                           |
+---------------------------------------------------+
```

- **左侧**: 数据源面板 (树形结构, 支持搜索, 字段可拖拽至画布)
- **中间**: 画布区域 (带标尺、辅助线, 支持缩放)
- **右侧上**: 结构树面板 (显示元素层级关系)
- **右侧下**: 属性面板 (根据选中元素动态切换)
- **底部**: 概览图 / 历史记录 / 动画面板

所有面板支持: 折叠、关闭、拖拽移动、调整高度, 通过数字键(1-9)快捷开关。

### 1.2 顶部"数据"物料面板

鼠标悬浮至顶栏"数据"按钮后展开 Grid 网格式下拉面板, 包含 40+ 种物料:

| 分类 | 物料 |
|------|------|
| 表格类 | 单元格、**数据表格**、自由表格、自由单元格、横纵卡片 |
| 富文本类 | 富文本(2种)、HTML |
| 媒体类 | 背景图、视频、轮播图、条件图片 |
| 数据展示 | 进度条、环形进度、评分、热度、标签云、信号灯、告警 |
| 计算类 | 公式、函数、标尺 |
| 批量生成 | 批量文本、批量BWIP、批量二维码、批量条形码、批量图片 |
| 交互类 | 签名、数字调节、触压按钮、链接、通知 |
| 嵌入类 | 地图、Iframe、PDF文件、时钟、天气、风机 |
| 打印控制 | 序号器、页码器、骑缝章、分页符、日期时间 |

**UI 特征**: 图标 + 文字, 4-6列网格排列, 支持拖拽至画布。

## 2. 数据表格核心设计

### 2.1 行区域模型 (Row Zones)

数据表格将行分为 **6 个功能区域**, 画布上通过左侧标签明确标注:

```
+------------------------------------------+
|  头 (Header)   | 页面级头部, 可放任意元素     |  <-- 可嵌套单元格/表格
|                | 如表单字段 {#姓名} {#性别}   |
+------------------------------------------+
|  标题 (Title)   | 列标题行                   |  <-- 序号 | 展现量 | 点击率
+------------------------------------------+
|  数据 (Data)    | 数据绑定行, 循环渲染         |  <-- {#点击量} | {#展现量} | {#点击率}
|                | "重复数据行" 标记            |
+------------------------------------------+
|  合计 (Summary) | 本页小计/汇总行             |  <-- 本页小计 | [计算:本页累计] | [计算:本页求和]
+------------------------------------------+
|  尾 (Footer)   | 总计行                     |  <-- 总计 | 这里放总计
+------------------------------------------+
|  留白 (Blank)   | 底部保留空白               |
+------------------------------------------+
```

**核心设计理念**:
- 将打印报表中常见的逻辑区域固化为表格结构的一部分
- "头"区域可放置任意元素(单元格容器), 不仅限于表格列
- "数据"区域是唯一会循环渲染的区域, 用 `{#字段名}` 绑定
- "合计"支持内置计算公式: `[计算:本页累计]`、`[计算:本页求和]`
- 每个区域都可以通过开关独立隐藏/显示

### 2.2 数据绑定语法

| 语法 | 用途 | 示例 |
|------|------|------|
| `{#字段名}` | 数据源字段绑定 | `{#姓名}`, `{#点击量}` |
| `[计算:公式]` | 内置计算表达式 | `[计算:本页累计]`, `[计算:本页求和]` |

## 4. 交互模式分析

### 4.1 选择层级机制

采用**逐层点击下钻**模式, 从外到内:

```
画布(页面) → 数据表格 → 单元格容器 → 格子 → 文本内容
```

- **第 1 次点击表格**: 选中整个数据表格, 属性面板显示表格级属性
- **点击表格内格子**: 选中格子所在行, 属性面板同时显示表格属性+格子属性(滚动查看)
- **通过结构树点击"单元格"**: 直接选中单元格容器, 显示单元格级属性
- **双击格子**: 进入文本内联编辑模式

**面包屑导航**: 底部显示选择路径, 可点击跳转至任意层级:
```
选中<数据表格> > 选中<单元格> > 选中<数据表格> > 模板<带头尾的数据表格>
```

### 4.2 表格编辑交互

**选中表格后的浮动工具栏** (表格上方出现):

```
[撤销][重做] | [行列操作组] | [对齐组] | [层级操作] | [更多操作 ···]
```

主要行列操作包括:
- 在前方/后方插入行
- 在前方/后方插入列
- 删除行/删除列
- 合并单元格 / 拆分单元格
- 行高/列宽调整

**右键上下文菜单**:
| 操作 | 快捷键 |
|------|--------|
| 克隆 | Cmd+D |
| 复制 | Cmd+C |
| 剪切 | Cmd+X |
| 删除 | Delete |
| 移至顶层 | T |
| 上移一层 | U |
| 下移一层 | D |
| 移至底层 | B |
| 组合 | G |
| 取消组合 | Shift+G (禁用态) |
| 全选所有同类 | Cmd+A |
| 全选区域同类 | Option+Cmd+A |
| 反选 | Cmd+I |
| 反选可移动 | Option+Cmd+I (禁用态) |
| 复制JSON数据 | - |

### 4.3 拖拽交互

- **物料拖入**: 从顶部"数据"面板拖拽"数据表格"至画布, 创建新表格
- **数据源字段拖入**: 从左侧数据源面板拖拽字段至表格格子, 自动绑定 `{#字段名}`
- **元素移动**: 选中元素后通过拖拽手柄 (十字箭头) 移动位置
- **行区域标签**: 画布左侧显示各行区域的文字标签(头/标题/数据/合计/尾/留白), 方便识别

### 4.4 视觉反馈

| 状态 | 视觉效果 |
|------|----------|
| 选中表格 | 蓝色虚线边框包裹整个表格, 显示 resize handles |
| 选中行 | 行背景高亮为浅蓝色 |
| 选中格子 | 格子边框高亮, 显示格子级操作按钮 |
| 数据行标识 | "重复数据行" 文字标签显示在数据区域下方 |
| 区域标签 | 左侧竖向显示: 头/标题/数据/合计/尾/留白 |

## 5. 数据源面板

### 5.1 结构

左侧数据源面板采用**树形结构**, 支持:

- **搜索**: 顶部搜索框过滤数据字段
- **多数据源**: 支持多个独立的数据集 (出差申请单、人物信息、物联网、报表字段等)
- **嵌套结构**: 数据源内支持对象嵌套和数组嵌套
  - 简单字段: 二维码信息、申请人、联系电话...
  - 子对象(可展开): 人员信息 → 出差人/科室/职务...
  - 子数组(可展开): 行程信息 → 出发地/目的地/日期...
- **字段图标**: 每个字段前有拖拽手柄图标 (::)

### 5.2 数据绑定方式

- 拖拽字段至表格格子 → 自动绑定
- 点击格子后手动输入 `{#字段名}`
- 支持复杂的数据结构: 表头数据(对象) + 列表数据(数组)

## 6. 结构树面板

选中数据表格时, 结构树显示:

```
数据表格
  ├── 文本
  └── 单元格
```

- 点击结构树节点 → 选中对应元素
- 点击节点旁的可见性/锁定图标 → 控制元素状态
- 支持拖拽排序元素层级

## 7. 关键设计决策总结

### 7.1 值得借鉴的设计

1. **行区域模型**: 将打印报表的 头/标题/数据/合计/尾/留白 固化为表格结构, 每个区域有明确语义, 左侧标签直观标注
2. **分页打印属性**: "每页标题"、"初始显头"、"最后显尾"等开关精准对应打印报表场景
3. **属性面板层级联动**: 选中不同层级展示不同属性, 底部面包屑支持层级跳转
4. **格子级精细控制**: 四边边框独立控制、跨行合并开关、溢出省略、显示格式等
5. **内置计算**: `[计算:本页累计]`、`[计算:本页求和]` 内置公式, 降低用户配置成本
6. **空行填充**: 数据不足一页时用空行填满, 保持打印版面整齐

### 7.2 潜在问题

1. **属性面板过长**: 表格级 + 格子级属性在同一面板中纵向排列, 需要大量滚动
2. **选择层级混乱**: "选中<数据表格> > 选中<单元格> > 选中<数据表格>" 出现了重复层级, 面包屑容易困惑
3. **双击才能编辑文本**: 需要两次操作才能编辑格子内容, 交互路径较长
4. **物料面板体量大**: 40+ 种物料在一个面板中平铺, 无分类 Tab/折叠, 查找效率低

### 7.3 与 EasyInk 对比要点

| 维度 | report-designer | EasyInk 可参考 |
|------|----------------|--------------|
| 行区域 | 6区域(头/标题/数据/合计/尾/留白) | 根据实际需求决定区域数量 |
| 数据绑定 | `{#字段名}` 文本模板 | 考虑更直观的字段选择器 |
| 属性分层 | 表格/单元格/格子 3层共用一个面板 | 可考虑 Tab 或折叠分组 |
| 边框控制 | 统一边框 + 单格四边独立 | 类似 Excel 的边框选择器 |
| 打印特性 | 深度适配(每页标题/空行填充/多栏打印) | 按需引入 |
| 宽度分配 | 占比分配 | 考虑支持固定宽度/自适应混合 |
